    What is a plugin

    The basic functions of the software is very simple, other extensions are provided in the form of plugins. Each relatively independent function will be made into a plugin, you can install different plugins to experience the corresponding function.

    Working mechanism

    The plugin will only execute on matching entries, and the execution conditions depend on the matching rules that you set. After the plugin has been executed, the contents of the entry (title, note, or tag) will be modified.

    Trigger timing

    Plugins can be triggered automatically or manually. Each plugin can be set independently whether to turn on automatic trigger.

    Automatic trigger

    When a new entry is added, all plugins that have the automatic trigger option are tried to execute in the order appears in the “My Plugin” list. If the entry matches the current plugin matching rules, the current plugin will be executed. After the execution, the next plugin will be executed. If it does not match, the current plugin will be ignored.

    Since the plugin will modify the contents of the entry after it is executed, whether the next plugin will be executed depends on the contents of the entry. Therefore, different execution orders may produce different results. By “drag and drop” you can sort “My Plugin” list to ensure the proper order of execution.

    Manual trigger

    On an existing entry, you have the option of manually triggering the execution of the plugin. Proceed as follows:

    1. Slide left on the entry
    2. Click the plugin button
    3. Select the plugin to be executed

    Manual trigger support batch operation, you can simultaneously select multiple entries, batch execution plugins.
