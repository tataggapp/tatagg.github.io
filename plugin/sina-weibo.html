Tatagg

    Features

    The soft will do customization support for the Weibo text and Weibo article. For Weibo text, the Weibo text of the content will be saved as a title to the entry. For Weibo articles, the article title will be saved to the title of the article, the article content is saved to the entry notes.

    The author’s name of the Weibo text and Weibo articles can be added as a tag to the entry. If you turn this feature on, you need to specify the parent tag of author’s name tag.

    Plugin execution conditions

    The webpage saved in the entry will be executed if the link is a Weibo text or a Weibo article, otherwise it will be ignored.
